HTML

  • The head element contains information about the webpage
  • The body element represents visible content shown to the user
  • You should only ever use heading "h1" once per webpage

CSS

  • Wild card selector * applies a rule to ALL elements visible on the page
  • The Box Model: Content>Padding>Border>Margin
  • Classes in CSS are designated with a period before the selector

Git

  • Move between branches with 'git checkout ----'
  • Check your current branch with 'git status'
  • Create a feature branch with 'git checkout -b feature/----- '
  • ~~~ GIT WORKFLOW ~~~
  • Step 1: git add -A
  • Step 2: git commit -m "your message here"
  • Step 3: git pull origin main
  • Step 4: git push origin feature/-------
  • Step 5: git checkout main
  • Step 6: git pull origin main

Javascript

  • A variable is a named container that allows us to store data in our code
  • Control Flow is the order in which a computer executes code in a script
